# 🎮 Dots and Boxes Game

A customizable two-player web game where players connect dots to complete squares and earn points.

![Game Screenshot](screenshot.png)

## 🌟 Features

- **Customizable Teams**: Change player names and colors for every game
- **10x10 Grid**: Classic dots and boxes gameplay
- **Real-time Scoring**: Track points as you complete squares
- **Beautiful UI**: Modern, responsive design with smooth animations
- **Cross-platform**: Works on desktop, tablet, and mobile devices
- **No Installation**: Play directly in your browser

## 🎮 How to Play

1. **Setup**: Enter team names and choose colors
2. **Take Turns**: Click between dots to draw lines
3. **Complete Squares**: Draw the 4th side of a square to claim it
4. **Earn Points**: Each completed square = 1 point
5. **Bonus Turn**: Complete a square and get another turn!
6. **Win**: Player with most squares wins!
